# BPSR-AutoMusic

AutoMusic es un script de para automatizar la reproducción de música en el juego mediante secuencias de teclas.

## Uso basico

Para ejecutarlo, simplemente descargas el .exe y lo abres.

> [!Note]
> Actualmente, solo estoy comprobando su compatiblidad con el "Keyboardist". Los demás instrumentos pueden no funcionar correctamente.

Lo único que tienes que hacer es elegir un instrumento y una canción, y el script se encargará de reproducirla:
- F9 para activar/desactivar el script.
- F10 para eliminar de memoria el script (El equivalente a cerrarlo con la X y tener que volver a abrirlo).

Podéis descargar la última versión aquí: https://github.com/Joseleelsuper/BPSR-AutoMusic/releases/latest

Para ejecutar el fichero .ahk necesitarás tener instalado [AutoHotkey v2.0.19](https://github.com/AutoHotkey/AutoHotkey/releases/tag/v2.0.19)
